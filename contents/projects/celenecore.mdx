# CeleneCore

CeleneCore is a professional Minecraft NeoForge (1.21+) plugin developed for **Celene Pixelmon**. It provides unique gameplay enhancements for competitive servers, including command utilities for sharing Pokémon stats and calculating power—designed for seamless integration, high performance, and reliability.

> **Note:** This project is primary use in Celene Pixelmon.

## Features

- **Share Pokémon Details:** Let players easily share selected stats or traits of their Pokémon with others on the server using customizable tags.
- **Pokémon CP Calculator:** Quickly calculate the Combat Power (CP) of any Pokémon in your party for perfect competitive prep and power balancing.
- **User-Friendly Commands:** Intuitive command set with tab-completion.
- **PlaceholderAPI Supported**
